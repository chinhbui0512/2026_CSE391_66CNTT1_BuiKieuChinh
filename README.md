## 📚 Thông tin môn học: CSE391 -- Nền tảng phát triển Web

### 🎯 Mục tiêu môn học

Môn học giúp sinh viên nắm được các công cụ và kiến thức nền tảng thường
dùng trong phát triển phần mềm và phát triển Web, bao gồm môi trường
phát triển, quản lý mã nguồn và các khái niệm cơ bản về hệ thống Web.


------------------------------------------------------------------------

### 💻 Kiến thức Web cơ bản trong môn học

Ngoài công cụ phát triển, sinh viên còn làm quen với kiến trúc Web và
các công nghệ frontend như: - HTML
- CSS
- JavaScript
- Frontend framework cơ bản
- Kiến trúc Web: Frontend -- Backend -- Database

------------------------------------------------------------------------

### 🧪 Hoạt động thực hành

Sinh viên thường thực hành thông qua: - Làm việc nhóm trên GitHub
- Tạo repository và quản lý source code
- Xây dựng trang Web cơ bản
- Thiết kế layout, form, validation bằng JS
- Tổ chức cấu trúc project chuẩn

Ví dụ các thao tác Git cơ bản: - git add
- git commit
- git push
- git pull

------------------------------------------------------------------------

## 🚀 Mục tiêu cá nhân khi học môn này

-   Hiểu rõ cách hoạt động của Web
-   Thành thạo Git và quy trình làm việc nhóm
-   Xây dựng được website cơ bản
-   Làm nền tảng cho học Web nâng cao và Backend sau này
